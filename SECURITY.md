# Segurança

Se encontrar uma vulnerabilidade neste projeto, **não** abra uma issue pública.

Envie os detalhes por canal privado aos mantenedores do repositório (por exemplo, e-mail ou security advisory do GitHub), incluindo:

- Descrição do problema
- Passos para reproduzir
- Impacto potencial
- Sugestão de correção (se houver)

Vamos confirmar o recebimento e trabalhar em um patch o mais rápido possível.
